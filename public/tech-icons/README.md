# Tech Stack Icons

Place your colored SVG icons in this folder.

## Required Files

Place these SVG files here:
- `node.svg`
- `react.svg`
- `stripe.svg`
- `supabase.svg`
- `mongodb.svg`

## Important Notes

1. **Use colored SVGs** - The app will automatically convert them to grayscale
2. **File names must match exactly** (lowercase)
3. **SVG format only** - Must be .svg files
4. **Size doesn't matter** - They will be rendered at 16x16px
